
## ⛅ Weather App React

A simple and modern Weather app built with React.  
Features include to be able to see the current weather(default on Tashkent), plus additional weekly weather forecast about the searched city. Designed to be a clean and minimal  UI.

## Features
- See the current weather
- Weekly forecast
- Modern UI
- Local state management with React useState

## Tech Stack
- React
- CSS (modern styles with hover, shadow effects)
- JavaScript (ES6+)

## Live Demo
[Try the app online](https://todo-app-react-gamma-weld.vercel.app/)

## Installation

1. Clone the repo:
  git clone https://github.com/AbdujabborovShaxriyor/weather-app-react.git 

2. Install dependencies:
  npm install

3. Run the app:
  npm start

## Usage
- Type a city name in the input box
- Click "search" to get the forecast for a week
- Click each day to get extra info about weather

## Note
Some parts of the CSS were AI-assisted. All functionality and React logic are fully implemented manually.
